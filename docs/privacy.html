Informativa sulla privacy

Ultimo aggiornamento: 16 luglio 2026

Rubrica Associati funziona senza account e senza server. Lo sviluppatore non raccoglie, trasmette, vende o condivide dati personali, dati di utilizzo o identificatori del dispositivo.

Dati memorizzati

I dati inseriti o importati — inclusi nomi, numeri di telefono, numeri tessera, date e note — restano nel database locale del dispositivo e sono protetti dal sandbox del sistema operativo.

Importazione, esportazione e chiamate

  • L’importazione legge esclusivamente il file scelto dall’utente.
  • L’esportazione crea un file locale che viene condiviso solo quando l’utente sceglie una destinazione.
  • Il comando di chiamata apre l’app Telefono; Rubrica Associati non registra né monitora le chiamate.

I file esportati non sono cifrati: devono essere condivisi solo con persone autorizzate e conservati in modo sicuro.

Servizi esterni

L’app non contiene pubblicità, strumenti di analisi o sistemi di tracciamento. I link al codice sorgente e alle donazioni si aprono nel browser e sono soggetti alle informative dei rispettivi siti.

Eliminazione dei dati

Gli associati possono essere eliminati dall’app. Disinstallando Rubrica Associati vengono rimossi anche il database e le impostazioni locali, salvo eventuali copie esportate dall’utente.

Contatti

Domande o segnalazioni possono essere inviate tramite GitHub Issues.
